การจัดการคลังวัสดุ (สูงสุด 5 คลัง)

การเชื่อมต่อฐานข้อมูล Google Sheets และการแจ้งเตือน

ใส่ Web App URL ที่ได้จากการ Deploy ใน Apps Script เพื่อเปิดการซิงก์ข้อมูลสองทิศทางแบบเรียลไทม์

วิธีการเชื่อมต่อแอปเข้ากับ Google Sheets (สำหรับผู้ดูแลระบบ)

  1. สร้างไฟล์ Google Sheets ใหม่ขึ้นมา 1 ไฟล์
  2. ไปที่เมนูการทำงานด้านบน: ส่วนขยาย (Extensions) > Apps Script
  3. นำโค้ดในไฟล์ [google-apps-script.js] ในโฟลเดอร์นี้ ไปคัดลอกวางทับแทนที่ของเดิมทั้งหมด
  4. กดปุ่มบันทึกและทำการสร้าง Deployment ใหม่ (New Deployment) เป็นแบบ เว็บแอป (Web App)
  5. สำคัญ: ในหัวข้อ 'ผู้เข้าถึง' ให้เลือกเป็น ทุกคน (Anyone) จากนั้นกด Deploy และอนุมัติสิทธิ์
  6. คัดลอก "URL ของเว็บแอป" (Web App URL) ที่ได้จากป๊อปอัปหลังจากกด Deploy (ลิงก์จะขึ้นต้นด้วย https://script.google.com/macros/s/...) *ห้ามนำลิงก์ในแถบเบราว์เซอร์หน้าแก้ไขโค้ด (Script Editor) มาใส่
